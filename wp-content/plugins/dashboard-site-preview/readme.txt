This plugin shows a preview of your site on the dashboard.

== Description ==

This plugin uses a simple iframe to display your site's front page in the dashboard. It was developed by 10-year-old Jesse Friedman.

== Installation ==

1. Upload `jf-dashboard-widget-site-preview.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How can I adjust the width and height of the iframe? =

We intend to make a settings panel with options in the near future. Now, however, you must edit the code to change width and/or height.
